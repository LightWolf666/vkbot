## Установка
1) Установим зависимости:
   
   Необходим Python 3.8:
   ```sh
   pip3.8 install vkbottle, requests, lxml, vk_api, beautifulsoup4
   ```
2) Установим Git Clone:
   
   Новейшая версия:
   ```sh
   sudo apt install git 
   ```
   
3) Сделаем Git Clone:
   
   Новейшая версия:
   ```sh
   git clone https://github.com/wezzyoffical/pumabot_lite.git
   ```
   
4) Запуск бота:
   
   Новейшая версия:
   ```sh
   python3.8 run.py
   ```
